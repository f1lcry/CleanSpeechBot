# BotSummarizer
Telegram-бот, который принимает голосовые сообщения, конвертирует их, транскрибирует с помощью локального Whisper, затем передаёт текст локальной модели Llama 3.1 8B (через Ollama) для форматирования, очистки и лёгкой суммаризации. После этого бот возвращает пользователю аккуратно оформленный текст.
